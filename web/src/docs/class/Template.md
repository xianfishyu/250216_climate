### setting-section

每一节的设置，可以不设置 `data-title`

- `data-title` - 标题

``` html
<setting-section data-title="标题">...</setting-section>
<setting-section>...</setting-section>
<setting-section>...</setting-section>
```

### setting-panel

选项容器

``` html
<setting-section data-title="标题">
    <setting-panel>...</setting-panel>
    <setting-panel>...</setting-panel>
    <setting-panel>...</setting-panel>
</setting-section>
```
